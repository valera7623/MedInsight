# Пациенты

## Зачем нужен раздел

Карточка пациента — центральная точка: к ней привязаны документы, прогнозы, DICOM-исследования и история.

## Список пациентов

1. Откройте **Пациенты** в меню.
2. Используйте **поиск** по ФИО, телефону или email.
3. Переключайте страницы кнопками «Назад» / «Вперёд».

**Скриншот:** таблица с колонками ID, ФИО, дата рождения, пол, телефон, email, кнопка «Карточка».

!!! note "Кого вы видите"
    Список зависит от вашей роли: врач видит своих пациентов и пациентов своего отделения; администратор — всех в клинике.

## Создание пациента

1. Нажмите **+ Новый пациент**.
2. Заполните обязательные поля: ФИО, дата рождения, пол, телефон, **отделение**.
3. Нажмите **Сохранить**.

**Зачем отделение:** определяет, кто из персонала видит карточку.

## Карточка пациента

Нажмите **Карточка** в списке или перейдите по ссылке `/patient/{id}`.

На карточке доступно:

- основные данные пациента;
- список загруженных документов;
- кнопки **Загрузить документ**, **Запустить прогноз**, **Экспорт PDF**;
- история прогнозов.

**Скриншот:** карточка с блоком «Документы», таблицей файлов и кнопками действий.

## Экспорт

- **PDF** — отчёт по одному пациенту (кнопка на карточке).
- **Excel** — список всех доступных пациентов (кнопка на странице «Пациенты»).

## Удаление

Удалять пациентов могут только **администраторы**. При удалении удаляются и все документы пациента.

## Частые вопросы

**Не вижу нужного пациента** — проверьте отделение и роль. Обратитесь к администратору для назначения отделения или флага «видеть всех пациентов».

**Исследователь видит странные имена** — это нормально: данные анонимизированы (`P-123 ANON`).
