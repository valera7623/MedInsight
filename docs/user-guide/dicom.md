# DICOM — медицинские изображения

## Зачем нужен DICOM

DICOM — стандарт медицинских снимков (КТ, МРТ, рентген). MedInsight позволяет загружать `.dcm` файлы, просматривать их в браузере и использовать метаданные в аналитике.

## Раздел DICOM

Откройте **DICOM** в главном меню.

**Скриншот:** сетка карточек исследований с превью, модальностью (CT/MR), датой и статусом «Готово».

## Загрузка исследования

1. Нажмите **+ Загрузить DICOM**.
2. Выберите **пациента**.
3. Перетащите файл `.dcm` в зону загрузки или выберите через диалог.
4. Нажмите **Загрузить**.

!!! info "Обработка"
    Большие файлы обрабатываются **асинхронно** (1–5 минут). Статус «Обработка» сменится на «Готово». Уведомление придёт через WebSocket или Telegram.

## Просмотр снимка

1. В списке нажмите **Просмотр** на готовом исследовании.
2. Откроется **DICOM-вьюер** (`/dicom/viewer/{study_uid}`).

### Возможности вьюера

| Инструмент | Назначение |
|------------|------------|
| Серии (слева) | Переключение между сериями снимка |
| Слайдер кадров | Навигация по слоям (КТ) |
| 🔍+ / 🔍− | Увеличение / уменьшение |
| ↻ | Поворот |
| W/L | Яркость/контраст (window/level) |
| Перетаскивание | Сдвиг изображения |

**Скриншот:** тёмная область просмотра с снимком, панель инструментов сверху, список серий слева, метаданные (Modality, Body Part) внизу.

## Фильтры

На странице списка:

- **поиск** по описанию и модальности;
- фильтр **модальности** (CT, MR, US…);
- фильтр **статуса** (Готово / Обработка / Ошибка).

## Ограничения

- Максимальный размер файла задаёт администратор (по умолчанию 500 МБ).
- Удаление исследований — только **администратор**.

## Безопасность

- Оригинальный DICOM **шифруется** на сервере.
- Для просмотра создаются PNG-кадры (без повторной передачи DICOM в браузер).
- Доступ только к пациентам вашей клиники.
