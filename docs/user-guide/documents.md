# Документы

## Зачем загружать документы

MedInsight автоматически извлекает из выписки:

- диагнозы;
- назначенные лекарства;
- текст для AI-прогнозов.

Без документов прогнозы будут менее точными.

## Поддерживаемые форматы

| Формат | Расширение |
|--------|------------|
| PDF | `.pdf` |
| Word | `.docx` |

## Загрузка с дашборда

1. На **Дашборде** нажмите **Загрузить документ**.
2. Выберите **пациента** из списка.
3. Укажите **тип документа**: выписка, лаборатория, история болезни.
4. Выберите файл или перетащите его.
5. Нажмите **Загрузить**.

**Скриншот:** модальное окно «Загрузить документ» с выпадающим списком пациентов и полем выбора файла.

## Что происходит после загрузки

1. Файл **шифруется** и сохраняется на сервере.
2. Celery ставит задачу **парсинга** (обычно несколько секунд).
3. Статус документа меняется: `uploaded` → `processing` → `parsed`.
4. При готовности приходит уведомление (WebSocket / Telegram, если настроено).

## Просмотр результатов парсинга

1. Откройте **карточку пациента**.
2. В таблице документов найдите нужный файл.
3. Статус **parsed** означает, что извлечение завершено.

Извлечённые данные используются в дашборде (топ диагнозов/лекарств) и в прогнозах.

## Скачивание оригинала

Через API (для интеграторов): `GET /api/documents/{id}/download` — расшифровка происходит в памяти, файл не сохраняется на диск в открытом виде.

## Безопасность

- Документы хранятся **зашифрованными** (age).
- Доступ только пользователям вашей клиники с правом просмотра пациента.
- Все загрузки записываются в **журнал аудита**.

## Ошибки

| Статус | Что делать |
|--------|------------|
| `processing` | Подождите 1–2 минуты |
| `failed` | Проверьте формат файла; загрузите снова |
| Ошибка 403 | Нет прав на пациента |
