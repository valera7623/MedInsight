# Начало работы

Руководство для врачей, медсестёр, исследователей и наблюдателей.

## Зачем нужен MedInsight

MedInsight помогает:

- хранить карточки пациентов в одном месте;
- загружать выписки и автоматически извлекать диагнозы и лекарства;
- просматривать DICOM-снимки в браузере;
- получать AI-прогнозы рисков реадмиссии и осложнений;
- видеть статистику по клинике на дашборде.

## Первый вход

1. Откройте адрес клиники, например: `https://medinsight.fileguardian.info/login`
2. Выберите **клинику (subdomain)** из списка.
3. Введите **email** и **пароль**.
4. Нажмите **Войти**.

**Скриншот:** страница входа с логотипом MedInsight, полями «Клиника», «Email», «Пароль» и кнопкой «Войти». Справа — переключатель темы (☀️/🌙).

## Регистрация

Если у вас нет аккаунта:

1. На странице входа нажмите **Зарегистрироваться**.
2. Заполните ФИО, email, пароль.
3. Выберите **роль** (см. таблицу ниже).
4. Для врача, медсестры или зав. отделением — выберите **отделение**.
5. Нажмите **Создать аккаунт**.

### Роли при регистрации

| Роль | Кому подходит | Что можно делать |
|------|---------------|------------------|
| **Врач** | Лечащий врач | Пациенты, документы, прогнозы, DICOM |
| **Медсестра** | Сестринский персонал | Просмотр отделения, экспорт |
| **Зав. отделением** | Руководитель отделения | Полное управление отделением |
| **Администратор** | IT/руководство клиники | Пользователи, отделения, аудит |
| **Исследователь** | Аналитика без ПДн | Просмотр анонимизированных данных |
| **Наблюдатель** | Аудит, контроль | Просмотр полных данных, экспорт |

!!! tip "Исследователь vs Наблюдатель"
    **Исследователь** видит данные **без ФИО и контактов** (анонимно). **Наблюдатель** видит **полные** карточки, но не может их изменять.

## Главное меню

После входа в верхней панели доступны разделы:

| Пункт | Назначение |
|-------|------------|
| **Дашборд** | Статистика, графики, прогнозы |
| **Пациенты** | Список и карточки |
| **DICOM** | Медицинские изображения |
| **Подписка** | Тариф и лимиты анализов |
| **Админ** | Только для администраторов |

**Скриншот:** верхняя навигация с активным пунктом «Дашборд», справа — кнопка «Выйти» и переключатель темы.
