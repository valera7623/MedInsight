Безопасность по умолчанию

MedInsight спроектирован для медицинских данных: шифрование, аудит, 2FA и compliance-ready архитектура.

  • Age-шифрование файлов на диске — расшифровка только в памяти
  • JWT access + refresh, блокировка после неудачных входов, TOTP 2FA
  • Полный audit log действий пользователей и экспорт в SIEM
  • DSAR runbook, retention policy, control mapping в документации
  • Demo mode с read-only middleware для безопасных демонстраций
